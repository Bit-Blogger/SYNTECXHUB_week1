# Student Performance Analysis

## Objective
Analyze student academic performance and identify factors affecting results.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset Features

- Gender
- Study Hours
- Attendance
- Sleep Hours
- Math Score
- Reading Score
- Writing Score

## Insights

- Attendance improves performance
- Study hours positively impact marks
- Adequate sleep supports better academic outcomes
